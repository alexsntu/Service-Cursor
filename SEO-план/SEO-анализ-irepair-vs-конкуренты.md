## ЧАСТЬ 2. КРИТИЧЕСКИЕ ПРОБЛЕМЫ (ПО ДАННЫМ)

### 🔴 ПРОБЛЕМА 1 — Страница /error/ занимает 8-е место в Google и получает 126 кликов

```
/error/  →  126 кликов  |  11 603 показа  |  CTR 1,1%  |  позиция 8,8
```

Страница с ошибкой индексируется Google на позиции ~9, собирает 11 тысяч показов в месяц и 126 кликов — пользователи приходят и видят ошибку. Это прямые потери клиентов и удар по поведенческим факторам.

**Исправление:** Настроить redirect 301 с `/error/` на главную или на `/catalog/`. Убрать страницу из индекса через `noindex` или Вебмастер.

### 🔴 ПРОБЛЕМА 2 — Главная страница: CTR 0,9% при 68 539 показах


Главная страница показывается почти 70 тысяч раз в месяц — но кликают меньше 1%. Нормальный CTR для позиции ~13 — 2–3%. Это означает, что Title и Description главной страницы непривлекательны в выдаче.

### ШАГ 16 — Оптимизация скорости загрузки (мобильные) 🔴
*Срок: 1–2 недели. Обновлено по данным PageSpeed Insights.*

**Полевые данные (реальные пользователи):** LCP 1,9 сек ✅ — зелёные  
**Лабораторный тест Lighthouse:** Score **43/100** 🔴 — критически плохо

| Показатель | Факт | Норма |
|---|---|---|
| FCP | 12,8 сек 🔴 | < 1,8 сек |
| LCP | 31,8 сек 🔴 | < 2,5 сек |
| TBT | 500 мс 🟡 | < 200 мс |
| Speed Index | 12,9 сек 🔴 | < 3,4 сек |
| CLS | 0 ✅ | < 0,1 |

**Корневые причины (найдены при анализе HTML):**

| Проблема | Детали |
|---|---|
| 29 из 31 JS-файлов блокируют рендеринг | Нет `defer`/`async` |
| HTML весит 492 КБ | Норма < 100 КБ. Калькулятор рендерит всё в HTML |
| Дубли скриптов | `script.js`, `script.min.js`, `script(1).js`, `script.min(1).js` — 4 версии |
| 3 библиотеки масок одновременно | `inputmask.min.js` + `inputmask.dependencyLib.min.js` + `imask.js` |
| 2 слайдера одновременно | `swiper.min.js` + `slick.min.js` |
| 0 preload-тегов | Браузер не знает, что грузить первым |

**Действия (для разработчика OpenCart):**
- [ ] Добавить `defer` ко всем скриптам кроме критичных (jQuery можно оставить)
- [ ] Добавить `async` к GTM и call.tracker: `<script src="..." async>`
- [ ] Удалить дубли: убрать `script(1).js` и `script.min(1).js` из шаблона
- [ ] Убрать один из слайдеров (оставить Swiper или Slick)
- [ ] Убрать 2 лишних библиотеки масок из 3
- [ ] Добавить preload в `<head>`: `<link rel="preload" href="/catalog/view/javascript/app.css?v=1.39" as="style">`
- [ ] Включить PHP OPcache и Gzip на хостинге
- [ ] Оптимизировать HTML калькулятора — загружать модели через AJAX, а не рендерить в HTML

**Ожидаемый эффект:** Score 43 → 70–80+, LCP 31,8 сек → 3–5 сек
